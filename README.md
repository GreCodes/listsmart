# 🧠 ListSmart

**AI-powered Etsy & eBay listing generator**

ListSmart turns product photos into high-quality marketplace listings in seconds using Google Gemini AI. It generates optimized titles, descriptions, tags, and pricing suggestions for resellers and small business owners.

---

## ✨ Features

- 📸 Upload product images (1–5 per item)
- 🤖 AI detects product details (brand, condition, materials, era)
- 🛍️ Generates Etsy + eBay optimized listings
- 🏷️ SEO-friendly titles and tags
- 💰 Suggested pricing range
- 📋 One-click copy for listings
- ⚡ Fast, browser-based workflow

---

## 🚀 How it works

1. Get a free Google Gemini API key  
2. Paste it into the app  
3. Upload product photos  
4. Add optional details (condition, category, price)  
5. Click **Generate Listings**  
6. Copy your optimized Etsy or eBay listing

## 🔑 API Key Setup

This app uses the Google Gemini API.

👉 Get your free API key here:  
https://aistudio.google.com/app/apikey

- Your API key is stored **only in your browser**
- It is never sent to any server except Google’s API

## 🤖 Gemini Free Tier (What you get)

ListSmart uses the Google Gemini API. The free tier is enough for testing and light use.

### ✍️ What you can do
- Generate text (titles, descriptions, tags)
- Answer questions and do calculations
- Analyze images (up to 1–5 images per request)

### 📸 Image support
- You can upload **1 to 5 images per request**
- The AI uses all images together to understand the item

### ⏱️ Usage limits (approximate)
- Around **50–150 requests per day**
- Around **2–5 requests per minute**

> Limits vary depending on your Google account and model version.

### ⚠️ Important
- The free tier is for small projects and testing
- Heavy usage may require enabling billing in Google AI Studio
- Each user must use their own API key
